# Personal Portfolio

Одностраничный портфолио‑сайт разработчика / AI‑интегратора с терминальным интерфейсом и фоновой «Матрицей».

## Стек

- Next.js 15 (App Router, SSR/SSG)
- React 19
- TypeScript 5
- Tailwind CSS 3
- shadcn/ui (Radix UI)
- Canvas API (анимация фона)
- Vercel (хостинг и деплой)

## Основные фичи

- Терминальный интерфейс с typewriter‑эффектами и ASCII‑артом
- Блоки с проектами (LLM‑боты, CRM‑боты, автоматизация)
- Секция «Обо мне» с фокусом на AI/автоматизацию
- Адаптивная верстка
